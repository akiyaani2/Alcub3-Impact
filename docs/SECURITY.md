# Security

If you find a security issue in ALCUB3 Impact, please report it privately first.

## How To Report

Email: `akiyaani2@gmail.com`

Please include:

- a clear description of the issue
- affected component or file path
- reproduction steps if available
- potential impact

## Please Do Not

- open a public issue with exploit details before disclosure
- include secrets, tokens, or private data in public reports

## Scope

Relevant areas include:

- the Next.js app in `app/`
- API routes and account flows
- data handling in `waterwatch/`
- deployment or configuration issues that expose sensitive behavior

We will acknowledge valid reports and work to remediate them responsibly.
